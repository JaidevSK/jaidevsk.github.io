PyVM.py: MIPS Simulator with Quantum Extensions
 
This module implements a MIPS simulator in Python, capable of executing
a subset of MIPS assembly instructions, including integer arithmetic, memory access,
control flow, floating-point operations (FPU), and system calls (syscalls).
It also includes extensions to simulate basic quantum computing operations
on a small number of qubits using NumPy.
 
The simulator uses byte addressing for memory and the program counter (PC),
starting code execution from address 0x00400000 by default or from the 'main' label.
It parses assembly files containing .text and .data segments, handling common
directives like .word, .float, and .asciiz.
